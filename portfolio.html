Help Interface

http://oit.semo.edu/content/Moodle/MoodleHelp.aspx

When I worked at Southeast Missori State Univerisity with the transition to the Moodle LMS, we needed a way to present help content to faculty. We had several categories each with three subcategories and some of our help articles fit into more than one subcategory. I built this help interface based on mockups designed by a co-worker. There are categories down the right side of the page. When you click a category, the page dynamically changes the subcategories with display below the selected article. Each article has a Guidesheet PDF and/or a tutorial video which open in modal dialogs.

Web Server

https://github.com/kevinphelps/WebServer

I wrote a web server program in my my Data Communications class to demonstrate my knowledge of the HTTP protocol. Beyond the requirements for this project, I included support for HTTP Keep-Alive (persistent connection), PHP rendering, request timeouts, etc.
