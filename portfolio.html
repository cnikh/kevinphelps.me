Help Interface

When I worked at Southeast Missouri State University with the transition to the Moodle LMS, we needed a way to present help content to faculty. We had several categories each with three subcategories and some of our help articles fit into more than one subcategory. I built this help interface based on mockups designed by a co-worker.

Link: Moodle Help page @ oit.semo.edu

Web Server

I wrote a web server program in my my Data Communications class to demonstrate my knowledge of the HTTP protocol. Beyond the requirements for this project, I included support for HTTP Keep-Alive (persistent connection), PHP rendering, request timeouts, etc. This project helped me to solidify my knowledge of the HTTP protocol by actually implementing a small part of it.
